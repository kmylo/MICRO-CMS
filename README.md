## Quick Start

```
1. git clone https://github.com/kmylo/MICRO-CMS-CHALL.git

2. cd MICRO-CMS-CHALL

3. yarn

4. yarn dev
```

## Atomic Design

Run the project's Storybook to see the atoms, molecules, organisms, templates, and pages,

>  `yarn run storybook`

Open [http://localhost:6006]
